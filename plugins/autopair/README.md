# zsh-autopair

Auto-closes brackets, quotes, and parentheses as you type.

- **Source:** [hlissner/zsh-autopair](https://github.com/hlissner/zsh-autopair)
- **Loaded by:** `engine/plugin-manager/plugin-manager.zsh` (deferred)
- **Installed to:** `~/.zsh/plugins/zsh-autopair/`

## Disable

Remove the `"defer" "hlissner/zsh-autopair"` lines from the ZPLUGINS array in `engine/plugin-manager/plugin-manager.zsh`.
