# zsh-autosuggestions

Shows ghost text suggestions from your command history as you type. Press → to accept.

- **Source:** [zsh-users/zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)
- **Loaded by:** `engine/plugin-manager/plugin-manager.zsh` (deferred)
- **Installed to:** `~/.zsh/plugins/zsh-autosuggestions/`

## Disable

Remove the `"defer" "zsh-users/zsh-autosuggestions"` lines from the ZPLUGINS array in `engine/plugin-manager/plugin-manager.zsh`.
