# fast-syntax-highlighting

Colors commands as you type — green for valid commands, red for errors.

- **Source:** [zdharma-continuum/fast-syntax-highlighting](https://github.com/zdharma-continuum/fast-syntax-highlighting)
- **Loaded by:** `engine/plugin-manager/plugin-manager.zsh` (deferred)
- **Installed to:** `~/.zsh/plugins/fast-syntax-highlighting/`

## Disable

Remove the `"defer" "zdharma-continuum/fast-syntax-highlighting"` lines from the ZPLUGINS array in `engine/plugin-manager/plugin-manager.zsh`.
